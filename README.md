🛒 DayCart – Smart Daily Essentials Delivery Platform

🚀 DayCart is a modern full-stack web application that automates daily essential deliveries through subscriptions and instant ordering. It aims to simplify everyday shopping by combining convenience, automation, and speed.

🌐 Live Demo

👉 

📌 Overview

DayCart is designed to solve the problem of repeatedly ordering daily essentials like milk, bread, fruits, etc. Users can subscribe to products and get them delivered automatically, or place instant orders when needed.

✨ Features
👤 User Features
🔐 Authentication (Login / Signup)

🔁 Subscription-based ordering (daily essentials)
⚡ Instant product ordering
📦 Order tracking system
🔍 Product search & filtering
🛠 Admin Features
📊 Dashboard for managing products

📦 Order management system

⚙️ System Architecture

Frontend handles UI & user interaction
Backend manages API logic & authentication
Database stores users, products, orders, subscriptions
